# Saudi Real Estate Transactions Analysis 2025

## Overview

This project presents an interactive Power BI report that analyzes real estate transactions in Saudi Arabia during 2025.

The report was developed using official open data published through the Saudi Open Data Platform (SDAIA) and provided by the Ministry of Justice under the dataset titled "Real Estate Transactions".

Quarterly datasets were collected, combined, cleaned, transformed, and analyzed to provide a comprehensive view of the Saudi real estate market throughout the year.

---

## Report Pages

### 1. Market Overview

* Total transaction count
* Total transaction value
* Number of active cities
* Number of active regions
* Quarterly transaction trends
* Activity distribution analysis

### 2. City Analysis & Geographic Distribution

* Geographic distribution of transactions
* Top-performing cities
* Transaction volume by city
* Transaction value by city
* City performance comparison

### 3. Real Estate Market Statistics

* Property classification distribution
* Properties per transaction analysis
* Area distribution analysis
* Quarterly comparison
* Statistical summary indicators

---

## Key Insights

* More than 196,000 real estate transactions were analyzed.
* Total transaction value exceeded SAR 200 billion.
* Riyadh and Jeddah recorded the highest real estate activity.
* More than 96% of transactions belonged to the single-property transaction category.
* Residential properties represented the largest share of transactions.
